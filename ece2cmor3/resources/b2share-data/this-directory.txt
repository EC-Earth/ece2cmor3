This directory need to contain the following files:

 nemo-vertices-ORCA1-t-grid.nc      ( 47M)
 nemo-vertices-ORCA1-u-grid.nc      ( 47M)
 nemo-vertices-ORCA1-v-grid.nc      ( 47M)
 nemo-vertices-ORCA025-t-grid.nc    (3,3M)
 nemo-vertices-ORCA025-u-grid.nc    (3,3M)
 nemo-vertices-ORCA025-v-grid.nc    (3,3M)
 fx-sftlf-EC-Earth3-T159.nc         ( 24K)
 fx-sftlf-EC-Earth3-T255.nc         ( 68K)
 fx-sftlf-EC-Earth3-T511.nc         (104K)

With a clone of the cmor-fixer one can download these files:
 git clone git@github.com:EC-Earth/cmor-fixer.git

The files can be found in the cmor-fixer repository in this directory:
 cmor-fixer/nemo-vertices

Now make links or copy these files from the cmor-fixer repository into this directory of ece2cmor.

One can as well manually download the files by using for instance:
 wget https://b2share.eudat.eu/api/files/359aeba7-9416-4896-bf34-6905eb21302e/fx-sftlf-EC-Earth3-T255.nc
